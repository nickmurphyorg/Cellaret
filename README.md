# Cellaret
An app for saving your favorite drinks. 

![Cellaret Hero Image](https://nickmurphy.org/wp-content/uploads/2020/02/Cellaret-Hero.jpg)

Users can scan the drink's barcode to download basic info about the drink. You can save the following information:
- Image
- Name
- Category
- Favorite?
- Alcohol / Volume

## Dependencies
This repository uses [CocoaPodsKeys](https://github.com/orta/cocoapods-keys) and an API key from [Barcode Spider](https://devapi.barcodespider.com).
